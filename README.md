# Gerador de Senhas

Este é um projeto em **Python** que gera senhas seguras com letras, números e símbolos. O projeto possui interface gráfica feita com **Tkinter**.

---

## Funcionalidades

- Permite escolher a quantidade de caracteres da senha (mínimo 8, máximo 20)
- Campo que aceita apenas números
- Caixa de mensagem com erro quando a quantidade mínima e máxima não correspondem
- Gera senhas aleatórias com letras maiúsculas, minúsculas, números e símbolos
- Exibe a senha na interface
- Copia a senha para a área de transferência automaticamente
